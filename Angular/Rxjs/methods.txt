
What is the use of pipe() in RxJS (Angular Context)
The pipe() method is used to combine multiple RxJS operators and apply them to an Observable in a clean, readable, and functional way.

Syntax:

observable.pipe(
  operator1(),
  operator2(),
  operator3()
)

Example:
this.http.get('/api/users').pipe(
  map(users => users.filter(u => u.active)),
  tap(data => console.log(data)),
  catchError(err => of([]))
)
.subscribe();

What Happens Step-by-Step:
HTTP request returns Observable
map() â†’ transforms data
tap() â†’ side effect (logging)
catchError() â†’ handles error
subscribe() â†’ executes stream

Common Operators Used Inside pipe() (Angular)
Very common in real Angular apps:
map() â†’ transform response
switchMap() â†’ API chaining / cancellation
debounceTime() â†’ search optimization
filter() â†’ condition filtering
takeUntil() â†’ unsubscribe handling
catchError() â†’ error handling
tap() â†’ logging/debug
